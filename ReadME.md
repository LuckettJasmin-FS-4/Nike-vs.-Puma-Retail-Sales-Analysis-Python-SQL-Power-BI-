📊 Nike vs Puma Retail Sales Dashboard
📌 Project Overview

This project analyzes retail sales data for Nike and Puma using SQL, data cleaning, and Tableau visualization. The goal is to uncover key insights into brand performance, product trends, and customer preferences.

🎯 Objectives
Compare total revenue between Nike and Puma
Identify top-performing and lowest-performing products
Analyze category preferences across customers
Build an interactive dashboard for business insights
🛠️ Tools & Technologies
SQL (SQLite) – Data querying and aggregation
Python (Pandas) – Data cleaning and preparation
Tableau Public – Data visualization and dashboard creation
GitHub – Version control and project hosting
📂 Dataset

The dataset includes:

Product details (brand, model, category, gender)
Pricing (base price, discount, final price)
Sales metrics (units sold, revenue)
Customer information (country, income level, rating)
📊 Key Visualizations
🔹 Brand Comparison

Compares total revenue between Nike and Puma to evaluate overall performance.

🔹 Top Products

Displays the highest-performing products based on revenue and units sold.

🔹 Lowest Products

Highlights underperforming products that may need improvement or removal.

🔹 Category Preference

Pie chart showing distribution of product categories (e.g., Running, Gym, Lifestyle).
